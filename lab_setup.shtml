Printers and Shared Drives

How to be a member of the SEL computer workgroup

In order to print to the SEL shared printers and use the S-drive, when you set up your computer, make sure that your computer is part of the workgroup 'ENGR-SEL'. This option is set in Control panel/System.

How to map the S drive*

At the SEL, we do not have a computer network, but we do share a hard drive that we call the 'Shared' or 'S'-drive. Please follow these instructions to map this drive in your computer:

  • Open Windows Explorer.
  • Go to menu 'Tools' and select 'Map Network Drive'.
  • Select 'S:' as the drive letter.
  • Make sure that the option 'Reconnect at logon' is checked.
  • Type "\\sel-server.me.wisc.edu\shared" (without double quotes).
  • Click OK.
  • Login: sel-server\seluser Password: sel13
  • Finally, create a new folder and give it the name that you want. This will be your folder in the shared drive.

*You don't necessarily have to be at a SEL computer to access this drive. As long as you are logged into the College of Engineering's network (like on a CAE computer, or using VPN, instructions on how to activate), you should be able to map to the shared drive as instructed above.

Do not use the shared drive as a permanent storage drive. Everybody in the SEL has access to the shared drive, so do not use to store sensitive information, and keep other copies of your documents.

How to add the printers

We have three printers: two black and white (Littlefoot and Bigfoot), and one color printer (Colorina).

Colorina (color printer)

First, download the driver from the Shared drive at S:\Colorina Driver\hp_cp3505x_driver.zip and uncompress it to a known directory (c:\temp\hp_cp3505x_driver is a good option).

  1. Open the start menu, highlight settings and then select "Printers and Faxes"
  2. Either double-click on "Add Printer" or select it from the list in the left hand margin of the opened page.
  3. Select Next at the first screen, and then make sure the button next to "Local Printer attached to this computer" is selected and that the check box is unchecked.
  4. Choose "Create new port" and then from the drop down box choose "standard tcp/ip port.
  5. A new window will pop up which you should choose next on.
  6. In the Printer Name or IP Address enter "colorina.me.wisc.edu" and select next.
  7. Select finish after it detects everything.
  8. When it asks for the driver, select "have disk" and navigate to where you uncompressed the files (choose the "cp3505" file) and select next.
  9. It will have you confirm that the driver is correct and select next.
  10. Set a printer name (colorina works), set if you would like it as your default printer and select next.
  11. Select next at the next screen.
  12. Choose whether you want to test printing and select next.
  13. If you tested printing it will ask whether it printed successfully (choose ok).

Littlefoot (big B&W printer)

To set up Littlefoot, you can do one of the two following methods. There is a better way and an easier way, although right now the only disadvantage I see to the easier way is that you can't actually call the printer 'Littlefoot', it gets called the name DOIT chose for it.

Either way you set up the printer, make sure to follow the last set of instruction on making 2-sided printing the default to help save paper.

Better Way

  1. To install the new printer, type the following into a Windows Explorer Address bar or open the link below with Internet Explorer
    \\Sel-server\shared\Littlefoot Driver\autorun.exe
  2. Wait for the page to load. Click 'Install Printer'
  3. Select 'English', click 'OK'
  4. Click 'Next'
  5. Select 'Connected via the network' and click 'Next'
  6. Select 'Basic network setup for a PC or server' and click 'Next'
  7. Select 'Specify a printer by address; and click 'Next'
  8. Select 'IP Address', enter 128.104.186.196, and click 'Next'
  9. Select 'Yes, install this printer' and click 'Next'
  10. Click 'Next'
  11. Select 'Minimum Installation', click 'Next'
  12. Under printer name, type 'Littlefoot', make sure that the default printer box is checked, and click 'Next'
  13. Select 'Not Shared', click 'Next'
  14. Type '1304 ERB' in the location box, click 'Next'
  15. Click 'Install'
  16. Click 'Finish'
  17. Click 'Close'

Easier Way

  1. In the Start menu, select 'Printers and Faxes' (this may be in the Control Panel on some computers)
  2. Double-click 'Add Printer'
  3. Click 'Next'
  4. Select 'A network printer …' and click 'Next'
  5. Select 'Connect to a printer on the Internet …' and type the following as the URL
  6. http://128.104.186.196
  7. Click 'Next'
  8. Under Manufacture, select 'HP', and under Printers, select 'HP LaserJet 4250 PCL 6' (it should appear at the top of the list, not alphabetically) and click 'OK'. If 'HP LaserJet 4250 PCL 6' does not appear, you can also use 'HP LaserJet 4000 Series PCL' and it will work the same.
  9. Select 'Yes' if you want to make this the default printer, and click 'Next'
  10. Click 'Finish'

Bigfoot (small B&W printer)

  • Go to menu Start, and select 'Printers'. Click on 'Add printer'
  • Select 'Add a network printer'.
  • In Windows XP, select 'Browse for a printer' and click next. Select '\\SEL-PRINTER\bigfoot' and click on Next. Click on 'Yes' in the warning messages. Repeat the process for Colorina.
  • In Windows Vista, the installation process fails. Please follow the instructions to install Littlefoot.

To make two-sided printing the default (Duplex mode)

  1. In the Start menu, select 'Printers and Faxes' (this may be in the Control Panel on some computers)
  2. Right-click on Littlefoot and select 'Printing Preferences'
  3. In the 'Finishing' tab, make sure the 'Print on both sides' box is checked, click 'OK'
